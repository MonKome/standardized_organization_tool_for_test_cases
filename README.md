📄 项目简介
    这是一个功能强大的 Excel 文件合并工具，旨在自动化处理来自多个源文件的 Excel 数据。它不仅能将多个文件的数据智能地汇总到一个工作簿中，还能保留源文件中的单元格合并样式，并自动应用统一的格式，极大地提升了数据整理的效率。
    该工具通过一个配置文件 (config.ini) 进行驱动，支持灵活的列名映射（包括精确匹配、近义词和模糊匹配），能够适应不同格式的源文件，最终生成一个格式规范、内容完整的汇总报告。
✨ 核心功能
    智能数据合并：自动扫描指定文件夹下的所有 .xlsx 文件，并将它们的数据合并到一个新的 Excel 文件中。
    灵活的列名映射：支持三级匹配策略，智能识别不同源文件中的列名：
    精确匹配：列名完全一致。
    近义词匹配：通过配置 synonyms 识别如“姓名”和“客户名”等。
    模糊匹配：利用 difflib 库处理拼写有轻微差异的列名。
    保留合并单元格：能够读取源文件中的合并单元格区域，并在合并后的文件中，根据数据偏移量，在正确的位置重新应用合并样式。
    自动样式应用：为生成的汇总文件自动应用统一的样式，包括：
        为所有单元格添加细边框。
        设置文本自动换行和垂直居中。
        根据内容长度自适应调整列宽。
    来源追溯：可选择性地在汇总数据中添加“来源文件”和“来源Sheet”列，方便追溯每一行数据的原始出处。
    详细日志记录：程序运行过程会生成详细的日志，记录文件处理、数据映射、错误警告等信息，便于调试和审计。
🛠️ 使用方法
    准备配置文件：在项目根目录下创建或修改 config.ini 文件，配置输入/输出路径、目标列名、列名映射关系等。
    准备源文件：将所有需要合并的 .xlsx 文件放入配置文件中指定的 input_folder 目录。
    运行脚本：在命令行中执行以下命令：  python merge_excel.py
    查看结果：程序运行成功后，会在配置文件中指定的 output_path 生成合并后的 Excel 文件。详细的运行日志会输出到 merge_log.log 文件中。
